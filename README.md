# 📊 Supply Chain Analytics and Inventory pipeline
## 📷 Dashboard Preview
![Dashboard Preview](supply chain and inventory.pdf)
End-to-end supply chain analytics project built using Python, MySQL, and Power BI. The project implements a complete ETL pipeline and interactive dashboard to analyze orders, profitability, logistics performance, and delivery risks with AI-driven insights.

---

## 🚀 Killer Features

### 💎 The "Big Picture" (KPIs)
* **💰 Money Maker:** Tracking a massive **$1.3M in Total Profit** at a glance!
* **📦 Order Surge:** Keeping tabs on **22K+ orders** flying through the system.
* **⏱️ Speed Demon:** Monitoring our **3.47-day average delivery**—because nobody likes waiting for packages! 🚚💨

### 🎨 Next-Level UI/UX (The "Vibe")
* **🌑 Midnight Mode:** A deep, dark-blue aesthetic that’s easy on the eyes and looks straight out of a sci-fi movie. 🌌
* **🍱 Bento-Box Layout:** Clean, rounded "tiles" (12px radius!) that keep everything organized and crisp. No clutter here! 📐
* **🌈 Neon Glow:** Using vibrant Cyans and Emerald Greens that practically jump off the screen. 🟢🔵

### 🔍 Deep-Dive Analytics
* **🗺️ Global Footprint:** An interactive map to see exactly where our fans are buying from. 🌎
* **🏆 Hall of Fame:** A "Top 10" list to see which products are bringing in the most 💵.
* **📊 Flow State:** Bar charts that show exactly where every order is—from "Pending" to "Delivered." ✅

### 🛠️ The Tech Magic
* **🧠 DAX Wizardry:** Custom formulas calculating profit-per-order so we know exactly where the gold is hidden. 🧙‍♂️
* **⚡ Super-Slicers:** Filter by **Year**, **Category**, or **Shipping Mode** with one click. It’s like a remote control for your data! 🖱️
* **🛡️ PDF-Proof:** Custom-built "Rectangle Hack" to ensure those dark colors stay dark, even when exported! 📑🚫⚪
